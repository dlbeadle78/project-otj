# Project OTJ Overview

## Product purpose

Project OTJ is an internal productivity toolkit for Health and Social Care Learning Coaches. It helps coaches build clear prompts that generate meaningful Off-the-Job learning activities for apprentices at Levels 2, 3, 4 and 5.

## Intended users

- Learning Coaches
- Assessors and tutors supporting Health and Social Care apprenticeships
- Internal colleagues reviewing or improving OTJ activity design

## The product is not

- A learner-facing assessment system
- A source of completed learner evidence
- A replacement for professional judgement
- A tool for inventing KSB codes or local workplace details

## Core principles

1. Activities must be realistic for adult care workplaces.
2. Prompts must support knowledge, skills and behaviours without over-claiming assessment coverage.
3. Outputs must use accessible, natural UK English.
4. Local policy, permission and role boundaries must be respected.
5. KSB links are guidance only unless verified against an approved source.
6. Learners must complete and reflect on their own learning.

## Current product areas

- OTJ activity prompt generator
- Level and topic selection
- Activity type and duration selection
- Service and learner-context options
- Generated ideas library
- Email-ready prompt output
- Starter prompt library

## Source of truth

Approved requirements, lists, prompts and design rules held in this repository take priority over older chat messages, screenshots or downloaded copies.
