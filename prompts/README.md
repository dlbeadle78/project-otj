# Prompt Library

This folder will hold the approved Project OTJ prompt library.

## Planned structure

- `level-2/`
- `level-3/`
- `level-4/`
- `level-5/`
- `shared/`

Each approved prompt should record:

- level
- topic
- activity type
- suggested duration
- learning purpose
- task focus
- level expectation
- method requirements
- output headings
- version or review date

Prompts must not provide completed learner evidence or invent KSB codes.
