# Project OTJ

AI-supported Off-the-Job learning toolkit for Health and Social Care Learning Coaches.

## Purpose

Project OTJ helps Learning Coaches create high-quality prompts for meaningful, apprenticeship-relevant Off-the-Job learning activities across Levels 2 to 5.

The toolkit is designed for Learning Coaches. It is not learner-facing and does not generate assessment evidence.

## Current scope

- Level-specific OTJ prompt generation
- Topic, activity type and duration selections
- Service and learner-context options
- Prompt library and generated ideas
- Learning Coach email output
- Shared design rules for future site development

## Repository structure

- `docs/` product, design and development documentation
- `prompts/` approved prompt libraries by level
- `data/` topic lists, activity types, durations and other structured options
- `src/` application source code
- `assets/` images and supporting files

## Working rule

This repository is the master project record. Approved requirements, prompt libraries and source files should be updated here rather than being held only in individual ChatGPT conversations.
